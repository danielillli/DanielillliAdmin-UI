# Admin Menu for FiveM QB-core

This script provides a comprehensive admin menu for managing various aspects of your FiveM server based on QB-core framework.

## Features
- Player Management
- Vehicle Management
- Server Management
- Teleportation
- Miscellaneous Actions

## Installation
1. Ensure you have QB-core installed.
2. Download the latest release from [GitHub](link).
3. Add the `adminmenu` folder to your resources directory.
4. Add `ensure adminmenu` to your server.cfg file.
5. Start your server.

## Usage
Once in-game, use the specified admin command to open the menu and access the various options.
